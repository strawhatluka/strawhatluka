## 🚀 Featured Projects

### [Cola Records](https://github.com/strawhatluka/cola-records)

Cross-platform desktop application for independent music label management. 243 commits, 15 releases, 153 React components, 39 backend service files, 12 IPC handler domains.

**Tech:** Electron, React, TypeScript, SQLite, IPC architecture
**Platforms:** Windows, macOS, Linux

### [Hytale Server Manager](https://github.com/strawhatluka/hytale-server-manager)

Desktop server management tool built with Electron 40 and React 19. 240+ tests, IPC architecture for secure main-renderer communication.

**Tech:** Electron, React, TypeScript, Jest, IPC architecture

### [Bwaincell](https://github.com/strawhatluka/bwaincell)

A comprehensive monorepo productivity platform with Discord bot (46+ commands), REST API (39 endpoints), and Progressive Web App for task management, reminders, notes, budgets, and scheduling.

**Tech:** TypeScript, Node.js, Discord.js, Express, PostgreSQL, Next.js, Docker, OAuth 2.0
**Deployment:** Self-hosted on Raspberry Pi 4B + Vercel
